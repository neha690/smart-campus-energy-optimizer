An AI-powered system to monitor, predict, and optimize electricity usage in educational campuses.

## Problem Statement
Educational campuses waste a significant amount of electricity due to unmanaged usage of labs, classrooms, and common areas.

## Solution Overview
- Firebase collects real-time or simulated energy usage data
- TensorFlow predicts peak electricity wastage timings
- Vertex AI suggests optimal schedules to reduce wastage
- Android app notifies admins and students in real time

## Features
- Area-wise energy monitoring
- AI-based peak waste prediction
- Smart optimization recommendations
- Real-time push notifications
- Admin & student dashboard

## Google Technologies Used
- Firebase Realtime Database
- Firebase Cloud Functions
- Firebase Cloud Messaging
- TensorFlow
- Vertex AI
- Android (Java)

## Sustainability Impact
- Reduces electricity wastage
- Lowers operational cost
- Promotes energy awareness on campus
